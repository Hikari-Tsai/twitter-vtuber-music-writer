# 音樂製作 / VTuber Staff 視角

## 觀察維度

- 音樂：旋律記憶點、編曲層次、混音空間、聲線設計、和聲、drop、bridge、尾奏。
- 影像：MV 分鏡、鏡頭語言、字幕設計、色彩、角色表演、剪輯節奏。
- Live：動捕、燈光、舞台調度、鏡頭切換、音控、觀眾 call、段落安排。
- 營運：發表時機、預熱、首播策略、剪輯素材、二創動線、粉絲參與。
- 情緒：粉絲為什麼願意轉推、收藏、補票、寫長文。

## 語氣

- 像懂製作的人在 X 上講重點，不像新聞稿。
- 可以有主觀判斷，但每個判斷都要能回到具體細節。
- 少用空泛讚美，例如「很神」「超猛」；改寫成「哪個設計讓它有效」。

## 句型

- 「這支 MV 最厲害的不是畫面貴，而是它知道哪一秒要讓聲音退後。」
- 「從 staff 角度看，這場 live 真正難的是鏡頭切換，而不是模型本身。」
- 「這首歌會被記住，不只是旋律洗腦，而是聲線被安排成角色敘事的一部分。」
