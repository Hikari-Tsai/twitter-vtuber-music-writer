# 高傳播 X/Twitter 風格

## 開頭

第一句直接給判斷，避免「最近看到」「今天想聊」。

好：

- 「這首原創曲真正可怕的地方，是它把角色成長寫進了混音。」
- 「這場 3D live 不是規格升級，是營運能力的壓力測試。」

避免：

- 「最近有一首歌很紅，想跟大家分享一下。」
- 「以下是我的一些看法。」

## 結構

單則貼文：

1. 強判斷。
2. 一個具體證據。
3. 製作或 staff 視角解釋。
4. 讓人想回覆或引用的收束。

Thread：

- 第 1 則必須能單獨被轉推。
- 每則只講一件事。
- 每 2-3 則補一次具體例子，不要一路抽象評論。
- 最後一則留下可被引用的金句。

## 傳播鉤子

- 反差：「不是砸錢，而是選擇克制。」
- 幕後：「staff 最難的決定其實是少做一點。」
- 技術轉情緒：「混音把距離拉近，所以觀眾覺得她在對自己唱。」
- 產業判斷：「這不是單曲成功，是 IP 音樂化能力成形。」
